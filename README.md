## How to donwload?
run this commands on terminal
``` 
cd Desktop
git clone https://github.com/theiskaa/hackernews_app
```
then you have to run pub get command
```
cd hackernews_app 
flutter pub get
```
And then you can run the application
```
flutter run
```
